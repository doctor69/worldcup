# ⚽ FIFA World Cup 2026 Simulator

Monte Carlo simulation of the FIFA World Cup 2026 — all 48 teams, real FIFA rankings (April 2026), 1000 complete tournament runs.

**Live site:** https://doctor69.github.io/worldcup

## What it does
- Simulates all 1,000 tournaments using real FIFA points, form, experience, and attack/defense metrics
- Full 48-team bracket: 12 groups → Round of 32 → R16 → QF → SF → Final
- Shows championship probability, final %, semi %, quarter %, R16 % for every team
- Group-by-group breakdown with head-to-head win probabilities
- Methodology tab explaining the math

## Stack
- [Astro](https://astro.build) (static output, GitHub Pages)
- React (interactive simulator component)
